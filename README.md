# mqfarooqi1.r-universe.dev

Registry for my personal [R-universe](https://r-universe.dev), which builds my R
packages into a CRAN-like repository at <https://mqfarooqi1.r-universe.dev>.

The packages are listed in [`packages.json`](packages.json). Install any of them
with:

```r
install.packages("dataProfilerR", repos = "https://mqfarooqi1.r-universe.dev")
install.packages("GSbench",       repos = "https://mqfarooqi1.r-universe.dev")
```
